## 仿美团的团购应用--

**环境要求**

1, 搭建好PHP+Apache+MySQL环境。

2，将hdgroup数据库导入。

3，由于网站用到了百度API功能，应在联网状态下访问。

4，在地址栏输入http://localhost/meituan 即可进入应用首页。

5，后台地址:http://localhost/meituan/index.php/admin
